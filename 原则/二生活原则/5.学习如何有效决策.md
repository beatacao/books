# 学习如何有效决策
* 日常生活中大多数决策是潜意识的：即无法准确描绘出来，让同样情况下的自己或其他人也能做到一样好
* 挑战好决策的可复制性

## 要认识到两点
* 影响好决策的最大威胁是有害情绪
* 决策是一个两步流程（先了解再决定）
* 不管一个选择看起来多么好，都要思考、提出疑问和探索真相后再做决定
* 做决策要考虑直接结果、后续结果和再后续的结果（不要只考虑直接结果，也不要不考虑直接结果）

### 把了解做好
```
对现实准确、完整认识需要两种能力：准确综合分析、综合考虑多个层级
1) 综合分析眼前的形势
2）综合分析变化的形势
3）高效地综合考虑多个层级
```
#### 综合分析眼前的形势
1、 不要陷在细节里
```
看到大局，综合分析，分清重要和不重要的事情
```
2、 信息来源的可信度：
```
最重要的决定之一是想谁或从哪里了解
```
3、 区分观点和事实：
```
不要听什么信什么，观点很廉价，很多人甚至会把观点当做事实描述（证据和求证）
```
4、 有时不急于做决定，可以放一放
```
所有东西都是放在眼前看很大。眼前正在发生的事情似乎很大，回头看则不然。跳出看全局，有时可以过段时间再决定
```
5、注重经过时间考验的经典，不过分夸大新东西

> 总结
* 要做好综合分析当下形势，就要有很好的宏观概念，不陷在细节中；做事或获取信息注重可信度和证据，不轻信；注重经过时间检验的经典，不过分夸大新事物；明白眼前的事情感觉上都会被放大，有时不急于做决定。

#### 综合分析变化中的形势
1、 始终记住改善事物的速度和水平
```
指定任何计划，要注重速度和应达到的水平
足够快的速度：生命中所有重要的东西需要以足够快的速度不断改善，超越平凡，走向卓越。
```
2、讨论常规，而不是例外；了解宏观概念时，例外不需要过分看重。
3、80/20法则
4、不做完美主义者：研究重要因素，过了一定临界点后，边际收益是有限的

> 总结
* 要做好综合分析变化中的形势，要注重速度，考虑重要的，普适性事物，懂得80/20法则，不要困在细节的完美中

#### 综合考虑各个层次
* 现实是分不同层次的，每个层次都能给你不同但有价值的视角
* 分析问题时，注意自己是在哪个层次上（基线以上还是基线以下的细节问题）
* 有意识地在各个层次之间转换
```
思考问题，先列出一条横轴主线；
接下来，如果针对主线上某一点有细节分支，在向下的数轴列出；
这样，有助于明确自己正在哪个层次上（基线之上还是细节中），并有助于及时拉回主线，把主线上的走完，不至于迷失在细节中而不自知
```


## 把决策做好（概率，统计，博弈）
```
通常存在两种决策方式：
1、以证据和逻辑为基础（较高层次的大脑）
2、以潜意识和情绪为基础（较低曾地的动物性的大脑）
决策做的不好，作为人类不会惨死，但一定会遭受到经济选择过程的惩罚
```

### 逻辑、理性和常识：综合分析现实、理解如何行动的最好工具，除此之外，不要依赖其他任何东西
```
卡尔.荣格所说：除非你意识到你的潜意识，否则潜意识将主导你的人生，而你将其称为命运。
```

### 根据预期价值计算做决策
* 凡事考虑两面性：好的价值*好的概率 - 坏的惩罚*坏的概率 = 正值
* 最好的选择是好处多于坏处的选择，不是毫无坏处的选择。不要发现一点问题就反对某件事，要权衡

### 机会成本
* 区分概率和可能性。万事皆有可能，重要的是概率，要把事情按照概率排序
* 区分重要的和想做的
* 对事情合理排序

最好的决策者：拥有理性、想象力和毅力，知道自己看重什么、想要什么，同时也利用计算机、算法和博弈论

理解事物的相关性，因果性

> 生活原则总结
拥有最好的生活，你必须：知道最好的决策是什么，有勇气做出最好的决策
