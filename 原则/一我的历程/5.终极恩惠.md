# 终极恩惠

* 坚持 -> 试错 -> 分析 -> 改进实践 -> 试错 -> ... -> 坚持...
```
尽管情况好多了，更稳定了，我们做的事却基本未变：与市场博弈，独立和有创意的思考如何押注，犯错误，把错误摆上桌面分析，找出根本原因，设计出新的更好的做事方法，系统化地落实这些更新，犯下新的错误，如此循环。
```

* 创新：发现‘通胀挂钩债券’ 和 ‘风险评价’
```
从作者公司这两次创新，我理解为：这种真正的、有价值的、可以引领和在一定程度上改变某一行业的创新，一定是在经过长期坚持，不断试错，分析的历程后，对行业有透彻理解后才可能有的，是对长期坚持的回馈。
可见，创新也要脚踏实地的从基础做起。
```

* 扩大还是坚持做精品店
```
作者认为：管理一家大企业的的难度，并不会比管理一家小企业更难；就像一个奥运选手面对的运动难度并不会比新手更小一样。尽管一个人会越做越好，但如果他想要更上一层楼的话，难度并不会降低。

启发：从这个角度出发，一个人如果停止进步，虽然过着普通的生活，但是任何小的困难可能都会如临大敌，甚至被压垮；而使自己越来越强，除了得到更好的生活之外，生活的难度并没有升高。
```

```
作者观点：我们能不能鱼和熊掌兼得，只不过是对我们创造力和性格的校验。

对于重要又看似矛盾的两件事，权衡后做出取舍二选一的做法有时可能是我们妥协的做法，或许应该尽力想一想，就像作者认为的，一定有我们还没有找到的办法，鱼和熊掌是可以兼得的。
```

```
如果确信某件事是正确的，值得做的，就选择去做。
困难和问题是永恒存在的，想办法，鱼和熊掌可以兼得。
```

* 不断充实原则
```
1、不断积累原则 -> 实践 -> 试错 -> 改进 -> 形成决策库 -> 企业文化（用于企业）/ 个人风格或个人魅力
2、关于在群体（或企业）内部如何很好额遵守规则，作者提到了透明度；让所有人更好的遵守规则的做法是保持透明度，使得大家理解所有事情，自己找出真相。没有透明性，人们往往会曲解各种事情，使之符合自身的利益，有时甚至是秘密这么做的，问题被隐藏起来，得不到解决。
3、基于透明性和开放性，可以就自己的行为和理由展开非常坦率的讨论，进而理解不同人的思维方式。在这一过程中，作者认识到：如果管理者不知道员工不同的思维方式，他就不知道员工将如何处理不同的情况，这就像一个工程领队不知道他的设备将如何工作一样。

启发：我们身边的人（尤其是重要的人），就像我们所处生活这台大机器的重要零件一样；为了让生活（工作）高效顺畅，就必须去了解身边的人，了解他们的思维方式及这些思维方式如何影响他们的行为；做到这些，才可以对生活（工作）这台大机器的高效顺畅运转有所掌控。
```
