# 前言
* 神经系统
```
在习练中使用辅助工具会产生适当的压力，使神经系统处于正确的平衡、节奏和和谐的振动之中，从而帮助关节、肌腱和纤维正确的调整归位。因为，神经系统是物质生理的身体和心理灵性的身体之间的媒介。
```
* 调整归位
```
瑜伽有助于我们在身心各个层面的连接和融合，并由此带来内外的平衡，也就是‘调整归位’（alignment）
真正的调整归位，意味着内在的意识融会贯通到身体的每一个细胞，每一根纤维。
经由同时关注瑜伽身体层面的锻炼和精神及灵性层面，真诚的瑜伽习练者将变得如一条平稳的河流，不断地灌溉和滋养着它所流过的每一寸土地。
‘没有人能够两次踏入同一条河流’，因此，每一次习练每一个体式都会让你获得崭新的能量，使生命更有活力。
```
